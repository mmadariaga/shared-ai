> **⚠ POST-HOC RECORD** — This proposal was backfilled after implementation. It describes a decision already made, not one being proposed.

## Why

SAI instruction files used vague, harness-agnostic references to "research subagent", "explore subagent", and "cheap research subagent" that did not map to any concrete skill. This made cost discipline rules ambiguous and inconsistent across instructions. Additionally, some instructions loaded full unified diffs unconditionally, wasting tokens on large changesets, and the `sai-explore` command was using an oversized model tier relative to its task class.

## What Changes

- `commands/claude/sai-explore.md` — model downgraded from `claude-opus-4-7`/`high` to `claude-sonnet-4-6`/`medium`
- `sai/instructions/accessibility.md` — "research subagent" references replaced with `budget-explorer` skill
- `sai/instructions/backfill.md` — conflict detection delegated to a `budget-explorer` subagent via structured prompt
- `sai/instructions/design.md` — "explore subagent" references replaced with `budget-explorer` skill
- `sai/instructions/implement.md` — `budget-subagent` skill references clarified and made explicit
- `sai/instructions/performance.md` — "research subagent" replaced with `budget-explorer`; added stat-first diff loading strategy
- `sai/instructions/remember.md` — cost discipline note updated to reference `budget-explorer`
- `sai/instructions/review.md` — "research subagent" replaced with `budget-explorer`; added stat-first diff loading strategy
- `sai/instructions/security.md` — "research subagent" replaced with `budget-explorer`; added stat-first diff loading strategy
- `sai/instructions/spec.propose.md` — cost discipline section simplified from 8 verbose rules to 6 rules that defer to the `budget-explorer` skill

## Capabilities

### New Capabilities

- `stat-first-diff-loading`: In review, security, and performance audits, check diff line count via `--stat` before loading the full diff; only load if ≤500 LOC, otherwise delegate per-file to `budget-explorer` subagents.

### Modified Capabilities

- `budget-explorer-subagent-binding`: All instruction files now reference the `budget-explorer` skill by name instead of vague harness-agnostic terms ("explore", "Explore", "cheap research subagent", "explorer custom agent").
- `sai-explore-model-tier`: `sai-explore` command downgraded from `claude-opus-4-7`/`high` to `claude-sonnet-4-6`/`medium`.
- `spec-propose-cost-discipline`: Cost discipline section in `spec.propose.md` simplified and unified — the detailed subagent spawn rules are now owned by the `budget-explorer` skill; the section retains only main-agent-specific rules.
