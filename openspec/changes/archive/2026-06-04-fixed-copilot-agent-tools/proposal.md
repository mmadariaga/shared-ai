## Why

Copilot agent definitions referenced non-existent tool names that did not match the current Copilot tool registry. This caused tool resolution failures when budget subagents attempted to invoke commands like `run/terminalCommand`, `read/directory`, and `read/fileContents`.

## What Changes

- `agents/copilot/budget-executor.agent.md`: Replaced `run/terminalCommand` → `execute/runInTerminal`
- `agents/copilot/budget-explorer.agent.md`: Replaced `read/directory` → `search/listDirectory`, `read/fileContents` → `read/readFile`
- `agents/copilot/budget-subagent.agent.md`: Replaced `read/directory` → `search/listDirectory`, `read/fileContents` → `read/readFile`, `edit/insertCodeBlock` + `edit/replaceSelection` → `edit/createDirectory` + `edit/createFile` + `edit/editFiles`, `run/terminalCommand` → `execute/runInTerminal`

## Capabilities

### New Capabilities

None — existing capabilities preserved with corrected tool identifiers.

### Modified Capabilities

- `budget-executor-terminal-execution`: Terminal command tool binding updated from `run/terminalCommand` to `execute/runInTerminal`
- `budget-explorer-file-discovery`: Directory listing tool binding updated from `read/directory` to `search/listDirectory`
- `budget-explorer-file-reading`: File content reading tool binding updated from `read/fileContents` to `read/readFile`
- `budget-subagent-file-discovery`: Directory listing tool binding updated from `read/directory` to `search/listDirectory`
- `budget-subagent-file-reading`: File content reading tool binding updated from `read/fileContents` to `read/readFile`
- `budget-subagent-file-editing`: File editing tool bindings replaced (`edit/insertCodeBlock` + `edit/replaceSelection` → `edit/createDirectory` + `edit/createFile` + `edit/editFiles`)
- `budget-subagent-terminal-execution`: Terminal command tool binding updated from `run/terminalCommand` to `execute/runInTerminal`
